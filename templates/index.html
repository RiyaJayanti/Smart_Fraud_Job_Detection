About This System

🕵️ Smart Fraud Detector

Smart Fraud Detector is an AI-powered tool that analyses job postings to identify potentially fraudulent listings. It combines multiple machine learning models, NLP-based text analysis, and collaborative filtering to flag suspicious job ads — helping job seekers stay safe from scams.

🔀 Hybrid Ensemble Model

Three models combined via Soft Voting:
• Decision Tree — interpretable, fast, rule-based
• Random Forest — robust, high accuracy (2× weight)
• Logistic Regression — stable probabilistic baseline (2× weight)

The hybrid outperforms any single model, especially on imbalanced data.

🤝 Collaborative Filtering

Truncated SVD decomposes the TF-IDF matrix into 50 latent semantic dimensions, capturing implicit patterns between job postings — similar to how recommendation systems find user-item similarities. These latent features are appended to the feature matrix before classification.

📊 Cross-Validation

Stratified 5-Fold CV preserves class ratios across folds, giving reliable, unbiased estimates of each model's performance. Results reported as mean ± std F1 score.

🔍 Explainable AI (XAI)

Feature importance from both Decision Tree and Random Forest reveals which words drive predictions. Per-prediction explanations multiply TF-IDF term weights × global RF importance to highlight the most influential tokens in each specific posting.
